# Craftlyt
Price it right. Sell it better.

# 3 Key MVP Features
1. Smart Pricing Engine
User uploads product (image + description)
System:
Scrapes / references similar items (basic API or dataset initially)
Suggests price range (min–max + recommended)
User can override

2. Simple Creator Storefront
Profile page with:
Products
Custom order toggle (“Accepts custom work”)
Instagram link

4. Request & Chat (Basic, not WhatsApp replacement)
Buyer sends:
Custom request (text + reference image)
Creator responds
